# Encoding

The encoding of the value of `data`. Defaults to `RAW` if not defined.

* `RAW` - RAW
* `BASE64` - BASE64
* `GZIP_BASE64` - GZIP_BASE64


## Values

| Name         | Value        |
| ------------ | ------------ |
| `Raw`        | RAW          |
| `Base64`     | BASE64       |
| `GzipBase64` | GZIP_BASE64  |
