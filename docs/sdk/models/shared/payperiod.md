# PayPeriod

The time period this pay rate encompasses.

* `HOUR` - HOUR
* `DAY` - DAY
* `WEEK` - WEEK
* `EVERY_TWO_WEEKS` - EVERY_TWO_WEEKS
* `SEMIMONTHLY` - SEMIMONTHLY
* `MONTH` - MONTH
* `QUARTER` - QUARTER
* `EVERY_SIX_MONTHS` - EVERY_SIX_MONTHS
* `YEAR` - YEAR
