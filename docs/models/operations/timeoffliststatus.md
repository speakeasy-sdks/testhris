# TimeOffListStatus

If provided, will only return TimeOff with this status. Options: ('REQUESTED', 'APPROVED', 'DECLINED', 'CANCELLED', 'DELETED')

* `REQUESTED` - REQUESTED
* `APPROVED` - APPROVED
* `DECLINED` - DECLINED
* `CANCELLED` - CANCELLED
* `DELETED` - DELETED


## Values

| Name        | Value       |
| ----------- | ----------- |
| `Approved`  | APPROVED    |
| `Cancelled` | CANCELLED   |
| `Declined`  | DECLINED    |
| `Deleted`   | DELETED     |
| `Requested` | REQUESTED   |
